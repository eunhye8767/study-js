04. 자바스크립트 시작하기 - 자바스크립트 워밍업
- 폴더 04_js_start
- 새버전 ES6 문법 경우 익스플로러 11에서 지원 안되는 것이 많다고 함.
  이 경우 바벨 같은 걸로 변환하는 환경세팅이 필요하다고 함. 
- 변수를 선언할 때만 var 이 아닌 새버전으로 변수 선언함.
  변수 let , const
  const 은 값이 변하지 않을 때
  let 은 값이 변할 수 있을 때 사용

- 변수의 scope 스코프(유효범위)
   var 경우 함수 단위로 결정이 되었음.
   var a = 100;
   function b() {
       c = 200;
       console.log(a);
   }
   b();
   100;
   c
   undefined;
   >> 함수 b() 안에서는 a 값을 알 수 있지만
   c의 값은 바깥에서 알 수가 없음.

- const, let의 유효범위는 똑같다.
  {} 블럭을 기준으로 결정이 된다.
  if (true) { 
    var a=100;
  }
  a
  100;
  >>  var 경우, 함수 단위로 구분되어 있기 때문에
  a 값이 출력이 됨. if문은 함수가 아님.

  if (true) { 
    let a=100;
  }
  a
  에러발생
  >> let, const 는 중괄호({}) 블럭으로 유효범위가 결정된다.
  >> var 과 let, const 차이!!
